## 🎯 项目定位

**XinBanAI V1** 是一个基于阿里云百炼 Qwen 大模型的**本地智能对话系统**，具备情感认知、长期记忆和定时任务等高级功能。

### 核心理念

- 🔒 **数据隐私优先**：所有数据本地存储，无云端通信
- 🧠 **情感认知能力**：四维情绪指标 + 五种状态切换
- 💾 **长期记忆管理**：SQLite + FAISS 双模检索
- ⏰ **自主任务调度**：AI 决策的延迟任务执行
- 🚀 **灵活部署**：支持云端 API（推荐）和本地推理（实验性）

## 💡 核心功能详解

### 1. 情感认知系统

#### 四维情绪指标

| 指标 | 含义 | 取值范围 |
|------|------|----------|
| 精力值 (Energy) | 精神状态 | 0-100 |
| 情绪值 (Emotion) | 情感倾向 | 0-100 |
| 专注值 (Focus) | 注意力集中程度 | 0-100 |
| 共鸣值 (Empathy) | 共情能力 | 0-100 |

#### 五种状态自动切换

| 状态 | 触发条件 | 语气特征 |
|------|----------|----------|
| 清醒 | < 5 分钟未对话 | 活泼主动 |
| 打盹 | 5-30 分钟 | 简洁慵懒 |
| 小憩 | 30-60 分钟 | 迷糊简短 |
| 午休 | 1-2 小时 | 被动回应 |
| 睡眠 | > 2 小时 | 极简/需唤醒 |

### 2. 长期记忆管理

#### 数据存储机制

- **SQLite 持久化**：8 张数据表存储结构化数据
- **FAISS 向量索引**：384 维向量用于语意检索
- **双模检索**：同时支持语意检索和时间检索

#### 8 大数据表

| 表名 | 用途 | 更新方式 |
|------|------|----------|
| `self_cognition` | 自我认知 | 增量更新 |
| `other_cognition` | 他人认知 | 增量更新 |
| `feelings` | 心情想法 | 每次对话 |
| `emotion_values` | 情绪数值 | 每次对话 |
| `long_term_memory` | 长期记忆 | 每次对话 |
| `event_summary` | 事件摘要 | 向量存储 |
| `user_info` | 用户画像 | UPSERT |
| `self_info` | 自我信息 | UPSERT |

### 3. 智能定时任务

#### 工作流程

```
AI 输出"下一次活动时间数值 [时间=XXX 秒]" 
→ 解析时间参数 
→ 启动守护线程 
→ 等待指定秒数 
→ 自动生成 Prompt 
→ 调用 AI 
→ 解析输出 
→ 写入数据库
```

#### 应用场景

- ⏰ 主动发起对话（如"该休息了"）
- ⏰ 周期性提醒（喝水、运动）
- ⏰ 延时反馈（任务进度跟踪）

### 4. 语意检索双模机制

#### 检索策略

```
graph LR
    A[用户输入] --> B[AI 首次回复]
    B --> C{是否包含<br/>语意检索指令？}
    C -->|是 | D[FAISS 检索 Top5]
    C -->|否 | E[直接输出]
    D --> F[再次调用 AI]
    F --> G[最终回复]
```

#### 优雅降级

- **首选**：FAISS 向量检索（毫秒级）
- **降级**：SQLite 时间检索（秒级）
- **保底**：无检索直接回复

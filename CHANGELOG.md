## [V1.0.0] - 2024-XX-XX

### ✨ 新增功能

#### 核心功能
- **情感认知系统**
  - 四维情绪指标（精力/情绪/专注/共鸣）
  - 五种状态自动切换（清醒/打盹/小憩/午休/睡眠）
  - 基于时间间隔的语气自适应调整

- **长期记忆管理**
  - SQLite 持久化存储（8 张数据表）
  - FAISS 向量检索（384 维 IndexFlatL2）
  - 语意关联和历史召回机制
  - 优雅降级策略（FAISS → SQLite）

- **智能定时任务**
  - AI 自主决策延迟任务调度
  - 支持 180 秒~8 小时灵活配置
  - 守护线程异步执行
  - 自动触发对话和提醒功能

- **增量认知适应**
  - 动态构建用户画像
  - AI 自我认知演进
  - UPSERT 增量更新机制
  - 避免重复记录

#### 技术架构
- **分层架构设计**
  - 用户交互层（CLI）
  - 功能层（ChatbotCore）
  - 读取层（DataReader）
  - 模块层（独立功能模块）

- **多入口分离**
  - `main.py` - 完整版启动程序
  - `db_viewer.py` - 数据库可视化
  - `cli_demo.py` - 命令行演示
  - `local_qwen_model.py` - 本地推理模型

- **模块化设计**
  - [`SQLiteDB`](sqlite_db.py) - 数据库封装
  - [`QwenAPI`](qwen_api.py) - 大模型调用
  - [`FAISSSearch`](faiss_search.py) - 向量检索
  - [`EventSummary`](event_summary.py) - 事件摘要

#### 用户体验
- **交互式 CLI**
  - 实时对话输入
  - 结构化输出展示
  - 情绪状态显示
  - 退出命令支持

- **数据库可视化**
  - Flask Web 界面
  - 表内容分页查看
  - 数据导出（CSV/JSON）
  - 实时统计信息

- **提示词优化**
  - 结构化 JSON 输出约束
  - 13 个功能字段完整定义
  - 增量更新原则
  - 容错降级机制

### ⚠️ 已知问题

#### 本地推理（实验性）
- Qwen2.5-7B-4bit 模型支持处于适配阶段
- 生成速度较慢（1.7-3.2 tokens/秒）
- 首次加载时间长（30-60 秒）
- 某些复杂推理任务准确率不如云端 API

#### 其他限制
- 主程序退出会导致守护线程终止
- 定时任务无法在主程序退出后执行
- FAISS 索引文件损坏需手动删除重建
- 向量维度固定为 384 维

## [4.x.x] - 2024-XX-XX

### ✨ 功能特性

- 基于 Qwen2.5-7B-4bit 量化模型（GGUF 格式）
- llama-cpp-python 推理引擎集成
- CPU 模式运行（无需 GPU）
- 基础对话功能正常

### 🔧 技术实现

- n_ctx=2048 上下文窗口
- n_threads=os.cpu_count() CPU 线程数
- n_gpu_layers=0 纯 CPU 模式
- Qwen2.5 官方标准 Prompt 格式

### 📊 性能表现

- 平均生成速度：1.7-3.2 tokens/秒
- 模型文件大小：约 4.3GB
- 内存占用：约 8GB
- 适用场景：中文对话、代码生成、知识问答
